# Server Side Rendering
Server side rendering (SSR) betyr at HTMLen genereres på serveren og sendes ferdig til nettleseren, i stedet for at nettleseren bygger siden selv.

## Hvordan starte applikasjon
1. Kjør `npm init` for å sette opp prosjektet
2. Installer Express med `npm install express`
3. Sørg for at PostgreSQL kjører på `127.0.0.1` (Localhost) med standard port (5432)
4. Kjør `node index.js`

## Hvilke teknologier er brukt
- Node.js
- Express
- PostgreSQL
